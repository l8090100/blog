# 作法
___
  
1.  麵粉:巧克力粉:植物油 = 3:1:2 (60:20:40)
2.  蛋四顆，蛋黃和60G牛奶打勻，蛋白和砂糖醋打發
3.  油加熱至有油紋，與過篩後的粉拌勻放涼，在和蛋黃液攪拌均勻
4.  分三次加入蛋白霜快速攪拌均勻後入模
5.  170度烤10分鐘，150度烤30分鐘
